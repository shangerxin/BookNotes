# Input output throughput 
-  Positional Parameters

first argument is $1, the second $2, and so on. The first positional parameter is not $0; this is the name of the shell or the shell script/command 

```
printf "Hello, %s!\n" "$1"
```
$ script john 
Hello john 

- Special *@#0$?_!- Parameters

$* and $@, expand to the value of all the positional parameters combined, and $# expands to the number of positional parameters. $0 contains the path to the currently running script or to the shell itself
