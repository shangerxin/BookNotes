# Understanding jobs in jenkins 
- create job 
- check this project is parameterised 
- Custom workspace path 

the job being executed is created by default in the ${JENKINS_HOME}\workspace folder. You will find ${JENKINS_HOME} usually at ${CurrentUser}\.Jenkins. You can change the workspace directory location by specifying the path in this field.

- you can trigger it remotely.

The format of this URL is: JENKINS_URL/job/<Job Name>/build?token=TOKEN_NAME.
JENKINS_URL: This represents the URL used to access the Jenkins instance (e.g., http://localhost:8080).

JobName: The name of a Jenkins job. Note that this should be Job Name and not Display Name.

TOKEN_NAME: This is an access token used to trigger this job. I added my authentication token.
