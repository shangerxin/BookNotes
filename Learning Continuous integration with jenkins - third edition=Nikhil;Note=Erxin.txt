# Enhancing your ci/cd pipelines 
- Configuring a Shared Library inside Jenkins

Open Jenkins, and navigate to Manage Jenkins | System.
On the resultant page, scroll all the way down to the Global Pipeline Libraries section and click on the Add button, as shown in Figure 12.5.
In the Name field, enter jenkins-email-lib.

- Using the Shared Library inside your pipeline

@Library('jenkins-email-lib') _
pipeline {
    agent any
    stages {
        stage('Example Stage') {
            steps {
                sh 'exit 1'
            }
        }
    }
    post {
        always {
            notifyByEmail 'recipient@example.com'
        }
    }
}
