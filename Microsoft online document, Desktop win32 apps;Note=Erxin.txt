## Kernel Transaction Manager
- reference 
https://docs.microsoft.com/en-us/windows/win32/ktm/kernel-transaction-manager-portal

The Kernel Transaction Manager (KTM) enables the development of applications that use transactions.

The KTM is used to implement Transactional NTFS (TxF) and Transactional Registry (TxR). TxF allows transacted file system operations within the NTFS file system. 

Managed and COM+ applications should use their native transaction managers.
